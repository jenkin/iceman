# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| <1.3    | :x:                |
| 1.X     | :x:                |
| 2.X     | :white_check_mark: |

## Reporting a Vulnerability

Please report any problems using GitHub's [Issues interface](https://github.com/jenkin/iceman/issues) with as detail as possible.
